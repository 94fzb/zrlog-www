### 3.3.0 (2026-02-01)

#### 新特

- 【博客页】多语言模板支持（beta），适配广受欢迎的 fluid 主题（仅支持 Native 包方式运行）
- 【博客页】预置 zrlog 程序主页主题（template-www）
- 【管理页】可设置链接（友情链接）和导航的图标

#### 优化

- 【管理页】登录页更直观便捷
- 【管理页】使用更大的圆角，在亮色模式下，顶部区域和侧边栏也使用主题色
- 【管理页】缓存更新更及时
- 【管理页】显示主题的更多信息
- 【插件】优化生成静态资源的逻辑
- 【构建】对 `Native Image` 包的架构统一定义，优化 native 包的启动速度
- 【开发】全局开发错误页的编码

#### 修复

- 【构建】 v3.2.2 windows 版本的 Native 包的 aot 映射不足的问题
