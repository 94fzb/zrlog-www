### 3.3.2 (2026-04-25)

#### 新特

- 【底层】Web 模块加载机制升级为 SPI 模式，统一支持按模块发现和装配 `admin`、`install`、`blog`
- 【管理页】AI 助手新增 Google Gemini provider，并支持流式返回内容

#### 优化

- 【底层】新增 `WebSetupContext`、`WebSetupProvider` 抽象，支持模块排序、禁用和重复 provider 保护
- 【管理页】补齐后台模块 SPI provider，统一管理端接入方式
- 【博客页】补齐博客模块 SPI provider，统一博客端接入方式
- 【编辑器】升级后台 Markdown Editor 到 `2.1.0`
- 【管理页】修复文章编辑页 AI 对话消息缓存的状态同步问题
