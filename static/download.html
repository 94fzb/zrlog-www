选择运行环境（模式）

Docker 模式运行（推荐） Linux amd64 FaaS.zip（无服务模式，已适配 AWS/Lambda[hello.handler]）（推荐） Linux arm64 FaaS.zip（无服务模式，已适配 AWS/Lambda[hello.handler]） Linux amd64.zip Linux arm64.zip（RPi/LinDroid） Debian/Ubuntu amd64.deb Debian/Ubuntu arm64.deb（RPi/LinDroid） Windows x86_64.zip macOS Intel.zip macOS Apple Silicon.zip 平台通用.zip（Java 版本 >= 11） 平台通用.war（Java 版本 >= 11 & Jakarta Web 容器）

系统要求

最低配置

  • Java 版本 (非必须)

    JDK 11 或更高版本

  • 数据库

    MySQL 5.6+ / MariaDB 10.0+

  • 内存

    至少 512MB 可用内存

  • 存储空间

    至少 200MB 可用磁盘空间

  • 操作系统

    Windows / Linux / macOS

推荐配置

  • Java 版本 (非必须)

    JDK 21 或更高版本

  • 数据库

    MySQL 8.0+ / MariaDB 10.5+

  • 内存

    1GB 或更多可用内存

  • 存储空间

    500MB 或更多可用磁盘空间

  • 操作系统

    Linux (推荐 Ubuntu)
