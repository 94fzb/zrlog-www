  • 【博客页】多语言模板支持（beta），适配广受欢迎的 fluid 主题（仅支持 Native 包方式运行）

Docker 一键启动 推荐

一条命令，即刻开始 

docker run -d -p 8080:8080 -v $(pwd)/conf:/opt/zrlog/conf --name zrlog zrlog/zrlog:3.3.0
然后访问 http://localhost:8080
