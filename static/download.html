ZrLog V3.1.0

发布时间：2024-07-19
  • Linux x86_64（无需 Java 环境）
  • Windows x86_64（无需 Java 环境）
  • macOS Intel（无需 Java 环境）
  • macOS Apple Silicon（无需 Java 环境）
  • 平台通用 (需 Java 版本 >= 21)

变更记录：

  • 服务端直接插入页面所需数据，大幅提升管理后台页面首次加载速度

  • 管理后台页面缓存机制，页面切换更流畅，告别等待

  • 将 Java 代码编译成原生可执行文件，启动速度大幅提升，运行效率更高

  • PWA 支持: 将博客写作添加到桌面或 Dock 栏，随时随地开启写作之旅

  • ...
初次部署 ZrLog，或者对 Java 不太熟悉的，可以参考这篇文章进行安装 https://blog.zrlog.com/run-zrlog-in-docker.html
